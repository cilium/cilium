Installation Guides
===================

These guides describes the various ways to install and configure Cilium in
different deployment modes. It focuses on a full deployment of Cilium within a
datacenter or public cloud. If you are just looking for a simple way to
experiment, we highly recommend trying out the :ref:`gs_guide` instead.
