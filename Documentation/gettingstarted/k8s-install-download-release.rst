.. note::

   First, make sure you have Helm 3 `installed <https://helm.sh/docs/intro/install/>`_.

.. only:: stable

   Setup Helm repository:

    .. code:: bash

        helm repo add cilium https://helm.cilium.io/

    .. parsed-literal::

        cd |SCM_ARCHIVE_NAME|/install/kubernetes
